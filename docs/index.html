Why canvasdown

Different by design

Features

Everything you need

Nine features that unlock use cases impossible with DOM-based renderers.

Streaming Render
canvasdown/stream
Append tokens one-by-one. Only re-paints changed lines. Built for LLM streaming.
Zero DOM Reflow
Core feature
Pretext measures text via canvas API — no getBoundingClientRect, no offsetHeight.
Node.js + PNG Export
canvasFactory
Inject @napi-rs/canvas for server-side OG image generation. No Puppeteer needed.
Text Along Path
renderTextOnPath
Render text following any Bezier curve. SVG has textPath — now canvas does too.
Per-Character Animation
animateText
wave · fadeIn · typewriter · bounce · explode — each letter animated independently.
Semantic Highlight
createOverlay
Highlight keywords without re-rendering. A separate canvas layer overlay.
Shrinkwrap Layout
shrinkwrap()
Binary-search the tightest container width that fits text. Figma-style auto-layout.
Text Fitting
fitText()
Shrink font size until text fits a container. No overflow, no truncation.
React Component
canvasdown/react
Factory function wraps canvasdown into a React component. ResizeObserver included.
Quick Start

Start in 30 seconds

Three lines for browser. Inject a canvas factory for Node.js. Stream tokens for AI chat.

Browser
Node.js
Streaming (AI)
import { render, darkTheme } from 'canvasdown'

const canvas = document.getElementById('output')
render(markdown, canvas, {
  width: 800,
  theme: 'dark',
  devicePixelRatio: window.devicePixelRatio,
})
import { exportPNG } from 'canvasdown'
import { createCanvas } from '@napi-rs/canvas'

const png = await exportPNG(markdown, {
  width: 1200,
  devicePixelRatio: 2,  // @2x — 2400px wide PNG
  theme: 'dark',
  canvasFactory: (w, h) => createCanvas(w, h),
})

// Save to disk
import { writeFileSync } from 'fs'
writeFileSync('og.png', Buffer.from(png.split(',')[1], 'base64'))
import { createStream } from 'canvasdown/stream'

const stream = createStream(canvas, { width: 680, theme: 'dark' })

// Connect to LLM (OpenAI, Anthropic, etc.)
const res = await fetch('/api/chat', { method: 'POST' })
const reader = res.body.getReader()

while (true) {
  const { done, value } = await reader.read()
  if (done) break
  stream.append(decode(value))  // only re-paints changed lines
}
stream.flush()
